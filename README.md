# 📅 日期倒數追蹤器 (Countdown Tracker)

一個輕量、美觀且具備響應式設計的網頁倒數計時器。支援同時追蹤多個重要日子（如跨年、考試、紀念日等），且資料會自動儲存於本地瀏覽器中，即使重整網頁或關閉瀏覽器，倒數依然在！

---

## ✨ 核心特色 (Features)

* **多組事件同時追蹤**：打破傳統計時器只能設定一組的限制，你想加幾個日子就加幾個。
* **純日期倒數顯示**：直觀顯示「剩餘天數」，當天會貼心提示「就是今天！」，若事件已過期也會自動顯示「已過期 X 天」並將卡片外觀變暗。
* **本地記憶功能 (LocalStorage)**：免註冊、免後端資料庫！所有設定直接儲存在你的瀏覽器中，無縫接軌每一次的訪問。
* **響應式網頁設計 (RWD)**：採用 CSS Grid 自動網格佈局，不論是用大螢幕電腦還是手機觀看，卡片排版都能流暢適應。
* **極簡輕量化**：完全使用純原生 HTML、CSS、JavaScript 打造，零外部套件依賴，秒速載入。

---

## 🛠️ 使用技術 (Tech Stack)

* **HTML5**：建構網頁基礎結構
* **CSS3**：使用 Grid & Flexbox 實現響應式排版與卡片美化
* **JavaScript (ES6+)**：負責時間邏輯計算與動態 DOM 操作
* **Web Storage API**：利用 LocalStorage 儲存使用者設定
